#
post请求实现文件导出

    因业务要求，需要在OA后台实现导出某些特定数据的功能。因为数据是在前台临时选中然后产出，所有无法使用url请求服务器资源模式。只能直接发送表格数据流到前台。

    项目里使用post进行数据交互。前端提交json数据。后端返回数据流。使用axios进行http请求。代码如下:

import axios from 'axios'

axios.post('exportUrl',//请求地址
    JSON.stringify({"data":'data'}), //相应的参数
        {   
            withCredentials: false,//因项目需要携带cookie，但是会异常，所以关闭改选项
            headers: {'Content-Type':'application/json'},//请求参数类型使用json
            responseType:'blob'//后端响应数据格式为blob类型。
        }).then(function (res) {
                //使用blob，将返回的数据流作为参数传入，生成本地文件。并指定文件类型。
                var blob = new Blob([res.data],{ type: 'application/x-xls' })
                var a = document.createElement('a');//创建一个a标签
                a.download = 'server.xls';
                a.href= URL.createObjectURL(blob)//将本地文件的作为a标签的文件指向。
                a.click();
            }).catch(function (error) {
                    console.log(error);
                });

    在开发过程中，最初使用json格式接收数据。但是转换后的文件会乱码。后改成指定类型为blob，才能正常被转换为文件。
